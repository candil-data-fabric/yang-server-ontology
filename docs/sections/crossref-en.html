Conventional Datastorec back to ToC or Class ToC

IRI: https://w3id.org/yang/server#ConventionalDatastore

One of the following set of configuration datastores: <running>, <startup>, <candidate>, and <intended>. These datastores share a common datastore schema, and protocol operations allow copying data between these datastores. The term "conventional" is chosen as a generic umbrella term for these datastores.
Is defined by
https://w3id.org/yang/server#
has super-classes
Datastore c
has sub-classes
Candidate Datastore c, Intended Datastore c, Running Datastore c, Startup Datastore c

Datastorec back to ToC or Class ToC

IRI: https://w3id.org/yang/server#Datastore

A conceptual place to store and access information. A datastore might be implemented, for example, using files, a database, flash memory locations, or combinations thereof. A datastore maps to an instantiated YANG data tree.
Is defined by
https://w3id.org/yang/server#
has sub-classes

is in range of
access datastore op, datastore op
